# Supply Chain Analytics Dashboard | Power BI

## Project Overview

This project presents a Supply Chain Analytics Dashboard developed using Power BI. The dashboard provides insights into supply chain performance, manufacturing quality, and product type trends. The dataset was sourced from Kaggle and transformed using Power Query and DAX to create meaningful business insights.

## Objectives

- Analyze supply chain performance metrics
- Monitor manufacturing quality and defect trends
- Evaluate product type performance
- Create interactive visualizations for decision-making
- Demonstrate data cleaning, transformation, and dashboard development skills

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
- Kaggle Dataset

## Data Preparation

The dataset was cleaned and transformed using Power Query.

Key steps included:

- Data validation and cleaning
- Handling missing values
- Data type corrections
- Creating calculated columns
- Developing DAX measures
- Preparing data for reporting and visualization

## Dashboard Pages

### 1. Supply Chain Overview

This page provides a high-level summary of supply chain operations through KPIs and performance metrics.

Key Insights:
- Overall supply chain performance
- Business KPIs
- Operational trends
- Performance monitoring

### 2. Manufacturing QC Analysis

This page focuses on manufacturing quality and process evaluation.

Key Insights:
- Quality control metrics
- Defect analysis
- Manufacturing performance
- Process monitoring

### 3. Product Type Analysis

This page analyzes product categories and their contribution to business performance.

Key Insights:
- Product performance comparison
- Category-level analysis
- Product trends
- Business contribution by product type

## Features

- Interactive dashboards
- Dynamic filtering and slicing
- KPI tracking
- Business performance monitoring
- Quality control analysis
